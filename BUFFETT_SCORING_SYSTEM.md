# 📊 纯数据驱动的巴菲特评级系统

## 核心原则
- 不用 LLM，完全基于财务数据
- 所有规则公开透明、可解释
- 遵循巴菲特的投资哲学（护城河、盈利能力、估值、安全边际）

---

## 评级指标体系

### 1️⃣ **护城河评分** (0-40 分)

#### A. ROE（股东权益回报率）- 衡量赚钱效率
```
超过 20% → 10 分  （优秀的护城河）
15-20%   → 8 分   （良好）
10-15%   → 5 分   （一般）
5-10%    → 2 分   （较差）
0-5%或负 → 0 分   （无护城河或亏损）
```

#### B. 净利率 - 衡量定价权
```
超过 20% → 10 分  （有定价权，如茅台）
10-20%   → 8 分
5-10%    → 5 分
0-5%     → 2 分
负数     → 0 分
```

#### C. ROE 稳定性（最近5年）
```
波动 < 10% → 10 分  （稳定）
波动 10-20% → 7 分
波动 20-50% → 3 分
波动 > 50% → 0 分   （不稳定）
```

#### D. FCF 质量（OCF/Net Profit）
```
> 0.8x   → 10 分  （利润质量好，收现能力强）
0.5-0.8x → 5 分
< 0.5x   → 0 分   （利润质量差）
```

**护城河小计：0-40 分**

---

### 2️⃣ **盈利增长** (0-30 分)

#### 最近3年利润 CAGR
```
> 30%   → 15 分  （高速增长）
15-30%  → 10 分
10-15%  → 7 分
5-10%   → 4 分
0-5%    → 1 分
< 0%    → 0 分   （衰退）
```

#### 利润增长的一致性（不要大起大落）
```
CAGR > 0 且无负增长年份 → 15 分
有1年负增长但总体正 → 8 分
有多年负增长 → 0 分
```

**增长小计：0-30 分**

---

### 3️⃣ **财务安全性** (0-20 分)

#### A. 负债率（债务/权益）
```
< 30%  → 10 分  （低风险）
30-60% → 7 分
60-100% → 3 分
> 100% → 0 分   （高风险）
```

#### B. 非流动负债比例（需要看完整数据）
```
< 40%  → 10 分
40-70% → 5 分
> 70%  → 0 分
```

**安全小计：0-20 分**

---

### 4️⃣ **估值** (0-30 分)

#### A. PE 相对历史中位数
```
< 0.8x 中位  → 15 分  （便宜）
0.8-1.0x    → 12 分
1.0-1.5x    → 8 分
1.5-2.0x    → 4 分
> 2.0x      → 0 分   （昂贵）
```

#### B. PB 相对历史中位数
```
< 0.8x 中位  → 15 分
0.8-1.2x    → 10 分
1.2-1.8x    → 5 分
> 1.8x      → 0 分
```

**估值小计：0-30 分**

| 总分 | 评级 | 结论 | 信息 |
|------|------|------|------|

| 20-29 | C- | 卖出 | 差公司 + 贵 |
