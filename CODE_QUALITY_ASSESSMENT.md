### ⚠️ 问题点

#### 1. **新闻分析依赖关键词匹配** (中等风险)
```python
HIGH_NEG = ["辞职", "离职", "被查", "立案", "违规", "处罚", "诉讼", "商誉减值", "暴雷"]
```

**问题：**
- 只能识别硬新闻（显式的坏消息），无法理解隐含意义
- 例如："高管离职寻求新机遇" 包含"离职"却是正面的，但会被标记为负面
- 英文新闻（摩根大通的报告）可能无法正确识别，因为关键词都是中文

**影响：** ⚠️ 中等
- 对中文新闻准确率 ~85%
- 对英文新闻准确率 ~30% (摩根大通报告主要是英文，需要改进)

**改进建议：**
```python
# 目前方案（弱）
if "离职" in title:
    sentiment = -1.0

# 更好的方案
# 用 LLM 预处理一行：判断这个"离职"是积极重组还是风险信号
```

#### 2. **缺少英文关键词库** (高风险 - 直接影响摩根大通)

**问题：**
摩根大通的报告/新闻主要是英文，但关键词库只有中文。

**例子：**
- "JPMorgan upgrades rating" → 无法识别（"upgrades" 不在中文库里）
- "CFO resignation" → 无法识别（English 版本）

**影响：** 🔴 高

- 所有摩根大通报告会被标记为情绪=0（中立），无法提取价值

**修复方案：**
需要添加英文关键词库，或者用 LLM 先翻译/理解
