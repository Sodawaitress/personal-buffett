# 📊 新闻量化分析系统升级

## 问题诊断
用户指出："只分析正负是否无法获得精确预测和动态"

**原问题**：
- ❌ 新闻仅分类为正/负/中性
- ❌ 无法量化信号强度
- ❌ 无法预测对股价的影响
- ❌ 无法识别信号动态变化

---

## 解决方案：量化新闻信号系统

### 核心改进

#### 1️⃣ 新增函数：`_analyze_news_signals()`
提取量化指标：
```python
{
    "sentiment_avg": float (-1 ~ +1)      # 平均情绪值
    "signal_count": {                      # 信号分布
        "high_neg": int,    # 高负面（质押、CFO离职等）
        "mid_neg": int,     # 中负面（减持、亏损）
        "mid_pos": int,     # 中正面（分红、签约）
        "high_pos": int     # 高正面（回购、创新高）
    },
    "key_signals": [...]               # 关键信号列表
    "impact_score": float (0-10)       # 影响力评分
    "momentum": str                    # 信号动态趋势
    "summary": str                     # 一句话摘要
}
```

#### 2️⃣ 量化指标说明
| 指标 | 含义 | 用途 |
|------|------|------|
| sentiment_avg | -1到+1 | 判断总体市场情绪方向 |
| signal_count | 各类信号计数 | 识别信号堆积（accelerating） |
| impact_score | 0-10分 | 预测对股价的实际影响 |
| momentum | positive/stable/negative | 判断趋势加速/平衡/恶化 |
| key_signals | 具体事件列表 | 护城河/管理层判断的依据 |

#### 3️⃣ 动态趋势识别
```
- accelerating_positive：正面信号>负面信号1.5倍 → 好消息持续 → 增持
- accelerating_negative：负面信号>正面信号1.5倍 → 坏消息堆积 → 减持/卖出
- stable：均衡状态 → 保持原判
```

#### 4️⃣ LLM Prompt 升级
新的 system prompt 明确指示 LLM：
- **第 2 点**（护城河）：必须参考「新闻量化信号」中的「关键信号」、「影响力评分」
- **第 3 点**（管理层）：新闻中的高影响力事件（CFO离职/回购）最关键
- **第 6 点**（催化剂）：新闻「情绪值」<-0.3 或「accelerating_negative」必须体现风险
- **Dims 维度**：各维度均需考虑新闻信号

---

## 数据流程

```
新闻列表
  ↓
_analyze_news_signals()
  ↓
量化指标（sentiment_avg, signal_count, impact_score, momentum）
  ↓
传入 LLM user_msg 的【新闻分析】区块
  ↓
LLM 基于量化数据生成护城河/管理层/动态趋势判断
  ↓
评级和结论（已包含新闻影响）
```

## 使用示例

### Intel (INTC) 的新闻分析
```
获取到 10 条新闻
情绪平均值：0.0 (中性)
影响力评分：1.0/10 (无重大信号)
动态趋势：stable
关键信号：(无)

→ LLM 判定：Intel 无重大催化剂，继续按基本面评估
```

## 关键词库覆盖

### 高正面信号（直接影响护城河/管理层）
- 回购、增持、大额分红、创历史新高、重大中标、获批上市

### 高负面信号（直接威胁）
- CFO辞职/离职、被查、立案、违规、处罚、诉讼、商誉减值、暴雷

### 中等信号
- 正面：派息、分红、签约、战略合作、净利润增长、获批、中标
- 负面：减持、亏损、下滑、下降、降级、失败、撤回、退出

### 噪音过滤
- 技术分析、资金流向日报、涨跌幅排名、K线、盘中播报

## 下一步优化方向

### 短期 (可立即实施)
- [ ] 扩展关键词库（行业特定词汇）
- [ ] 分市场关键词调整（A股 vs 美股 vs 港股）
- [ ] 添加新闻时间权重（近期新闻权重更高）

### 中期 (1-2周)
- [ ] NLP 情绪分析模型（替代关键词）
- [ ] 新闻来源权重（官方公告 > 分析师 > 社交媒体）
- [ ] 信号强度学习（根据历史股价反应自动调整权重）

### 长期 (1月+)
- [ ] 时间序列分析（信号峰值检测）
- [ ] 跨市场信号对比（同一公司美股vs港股新闻联动）
- [ ] 预测模型（基于historical signals预测7天收益率）
