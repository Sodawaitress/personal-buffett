# 巴菲特投资分析模型
*股票雷达 · 自选股评估框架*
*来源：历年伯克希尔股东信 + The Essays of Warren Buffett（Cunningham编）*

---

## 一、核心哲学

> "价格是你付出的，价值是你得到的。"
> "时间是优秀公司的朋友，是平庸公司的敌人。"
> "在短期，市场是投票机；在长期，市场是称重机。"
> "单一最重要的投资决策：这家公司有没有定价权。"

巴菲特的出发点永远是：**如果我拥有这整门生意，我愿意吗？**

---

## 二、四个投资前提（缺一不可）

1. **我看得懂这门生意**（能力圈）
2. **有持久的竞争优势**（护城河）
3. **管理层能干且诚信**（人）
4. **买价合理**（安全边际）

---

## 三、生意分三类（2007年股东信）

### 🏆 优秀生意（GREAT）
- 用极少的资本，持续产生极高的回报
- 几乎不需要再投入就能维持竞争地位
- 代表：See's Candy——1972年花$2500万买，35年只追加$3200万投入，累计税前利润超$20亿

**判断标准**：ROE长期 > 20%，且资本需求小

### ✅ 良好生意（GOOD）
- 为客户创造真实价值，竞争地位稳固
- 但需要持续的大量资本投入才能维持增长
- 代表：FlightSafety——投入$5.09亿资本，增加净利润$1.59亿（回报合理但不惊艳）

**判断标准**：ROE稳定 10-20%，需要持续再投入

### ❌ 糟糕生意（GRUESOME）
- 快速增长，但需要大量资本，最终几乎不赚钱
- 代表：航空业——"如果有一个有远见的资本家在莱特兄弟起飞那天，最好把飞机打下来"

**判断标准**：高增长 + 高资本消耗 + 低利润 = 价值毁灭机器

## 四、护城河五种类型

| 类型 | 描述 | 判断问题 |
|---|---|---|
| **品牌定价权** | 能涨价，客户还买单 | 提价10%会失去多少客户？ |
| **转换成本** | 换掉它太麻烦 | 客户换竞品需要付出什么？ |
| **网络效应** | 用的人越多越有价值 | 每个新用户让旧用户获益吗？ |
| **成本优势** | 比对手更便宜且有利润 | 有钱的竞争对手能在10年内复制吗？ |
| **有效规模** | 市场只容得下一两家 | 新进入者会破坏所有人的回报吗？ |

**护城河的关键**：是在变宽还是变窄？所有护城河都在变化，只是你不一定看得见。

**红旗**：如果护城河依赖某个天才CEO而非结构性优势，那不是护城河。

## 五、管理层评估

### 巴菲特的三不谈（2018年股东大会）

**第一把尺：生意经营好不好**
- 对比同行，5-10年内市场份额和ROIC怎么变的？
- 逆境（衰退期）怎么应对？逆境比顺境更能看清管理层

**第二把尺：对股东好不好**
- 读代理声明（proxy statement）：薪酬和业绩挂钩吗？
- 资本分配历史：并购价格合理吗？低价时回购了吗？

### 机构惰性（Institutional Imperative）——最被低估的红旗

巴菲特：这是"我在商学院没学到的最重要的事"。

四种症状：
1. **惰性**：无论方向对不对，组织会抵制任何改变
2. **吃饱资本**：有多少闲钱，就会有多少项目冒出来吸收，不管回报好不好
3. **马屁文化**：CEO的任何想法，下面都会找出数据支持
4. **同行模仿**：别人做收购，他也做；别人涨薪，他也涨

**怎么识别**：公司是否在周期高点做大并购？是否跑去做"转型"收购？薪酬是否跟行业同涨而非跟业绩挂钩？

### 管理层信号清单

| 信号 | 方向 |
|---|---|
| 大规模真金白银回购（低价时）| ✅ 强正面 |
| 稳定派息且逐年提升 | ✅ 正面 |
| 股价低迷时增发股票 | ❌ 负面 |
| CFO/CEO突然辞职原因不明 | ⚠️ 强警惕 |
| 高峰期大额并购不相关业务 | ❌ 红旗 |
| 频繁"一次性费用" | ❌ 红旗 |
| 管理层大幅减持 | ❌ 红旗 |

## 六、关键财务指标

### 自有收益（Owner Earnings）——巴菲特唯一的量化公式（1986年信）

```
Owner Earnings = 净利润 + 折旧摊销 - 维持性资本开支
               = 经营现金流 - 维持性资本开支
```

关键区分：**维持性资本开支**（保住现有地位必须花的钱）≠ **增长性资本开支**。
GAAP利润会失真，Owner Earnings更接近真实赚钱能力。

### ROE——最重要单一指标

巴菲特1987年信引用《财富》研究：1000家大公司中，只有**25家**连续10年ROE > 20%且无一年低于15%。这25家中，**24家**跑赢标普500。

- 优秀：连续5年 > 15%，不靠高杠杆
- 合格：稳定 10-15%
- 警惕：< 8%，或大幅波动

### 留存利润检验（每$1留存应创造 > $1市值）

```
检验 = 5年市值增加 ÷ 5年留存利润
结果 ≥ 1.0x = 管理层在创造价值
结果 < 1.0x = 管理层在毁灭资本，应该把钱还给股东
```

巴菲特案例：可口可乐1980-1987，每$1留存创造$4.66市值。

## 七、A股特殊调整

### 与美股的主要差异

| 挑战 | 应对方法 |
|---|---|
| 大股东利益与小股东不一致 | 重点看关联交易、大股东减持 |
| 会计质量风险 | 优先看经营现金流，对比净利润；5年现金流/净利润比值 |
| 政策壁垒≠真实护城河 | 问：政策一变，护城河还在吗？ |
| 国企管理层不为股东服务 | 参考巴菲特中石油模式：只在估值极度低估+可见现金流时买 |
| 商誉风险 | 频繁收购的公司（如复星）要核查商誉金额 |

### 巴菲特如何投中国股票

**中石油（2002-2007）**：
- 整个公司市值约$350亿，巴菲特估算内在价值约$1000亿
- 吸引点：可见的石油储量 + 3倍PE + 45%分红率（买入时现金收益率~15%）
- **他买的是确定性资产被极度低估，不是赌油价**
- 5年后$2750亿市值卖出，赚7.9倍

**BYD（2008）**：
- 由芒格主导，看中CEO王传福的技术执行力（真正的工程师型管理者）
- 电池技术护城河 + 即将到来的中国EV市场 + 可信赖管理层

**结论**：A股投资用巴菲特框架，要更严格筛选管理层诚信，更重视现金流质量，对国企用"中石油模式"（极度低估才进），对民企优先考察大股东行为。

## 八、自选股巴菲特评分卡

*基于2020-2024年财务数据，每季度更新*

### 🏆 阳光电源（300274）— A级·优秀生意

| 指标 | 数据 |
|---|---|
| 5年平均ROE | ~26%（2023年峰值41%）|
| 净利率趋势 | 10%→14%，持续上升 |
| 净利润增长 | 2020-2024累计约10倍 |
| 生意类型 | **优秀（GREAT）**：逆变器技术+储能规模优势 |
| 管理层信号 | 无重大负面 |
| 留存利润检验 | 待核算，方向极好 |

**巴菲特评语**：这是自选股里最接近"优秀生意"标准的公司。ROE持续超过20%，净利率逐年提升，说明竞争地位在加强而非侵蚀。储能市场30-50%增速如果属实，当前估值可能仍合理。**核心风险**：补贴依赖（政策护城河而非真实护城河）、中美贸易摩擦。

---

### ✅ 三一重工（600031）— B+级·良好生意（周期中）

| 指标 | 数据 |
|---|---|
| 5年ROE | 高峰29%→低谷6.7%→当前8.5%，周期波动 |
| 净利率趋势 | 2022年低点恢复，7.8% |
| 生意类型 | **良好（GOOD）**：周期性制造，但有品牌护城河 |
| 管理层信号 | ✅ 2026-04-02 真金白银回购13亿 |

**巴菲特评语**：典型优质周期股。巴菲特自己很少买纯周期股，但他会在周期底部关注管理层回购——这正是三一现在做的。海外64%收入分散了中国地产周期风险。**现在的位置**：还在复苏期，ROE尚未回到历史正常水平。继续持有逻辑成立，不是加仓时机。

---

### ⚠️ 复星医药（600196）— C级·观察

| 指标 | 数据 |
|---|---|
| 5年ROE趋势 | 12%→10%→9%→5%→6%，持续下滑 |
| 净利率趋势 | 13%→8.5%，下行 |
| 生意类型 | **良好偏糟糕**：收购太多，业务复杂，资本吃掉太多 |
| 管理层信号 | ⚠️ CFO刚辞职（2026-04-02）|
| 机构惰性风险 | 高——频繁收购，扩张积极 |

**巴菲特评语**：ROE连续5年下滑是结构性问题，不是暂时的。说明每一轮资本再投入的回报在降低。CFO辞职是近期最需要关注的信号——财务负责人离职往往早于坏消息公告。**不加仓，等公告**。

---

### ⚠️ 中金公司（601995）— C级·周期依赖

| 指标 | 数据 |
|---|---|
| 5年ROE趋势 | 14%→14%→9%→6%→5.5%，连续下滑 |
| 净利润 | 2022、2023、2024三连降 |
| 生意类型 | **良好偏糟糕**：护城河极弱，赚钱靠市场脸色 |

**巴菲特评语**：巴菲特几乎从不长期持有券商。原因：券商的"竞争优势"会随市场周期完全消失。年报净利+72%是因为基数极低，不是能力提升。ROE已降至5.5%，连存款利率都跑不赢。

---

### ❌ 北摩高科（002985）— D级·警报

| 指标 | 数据 |
|---|---|
| 2024净利润 | **-92.57%** |
| 2024 ROE | **0.62%**（2021年18.7%）|
| 2024收入 | 5.38亿（峰值11.32亿一半）|
| 生意类型 | **不确定**（军工信息不透明）|

**巴菲特评语**：这不是周期波动，这是基本面崩塌。净利润-93%，ROE从18%跌到接近0，收入腰斩。巴菲特原则：**不懂就不要碰，更不要在不懂的时候持有一个基本面急剧恶化的仓位**。军工信息不透明更是他明确回避的类型。**这只股票需要认真复盘持仓逻辑。**

## 九、日报分析模板

每条新闻用以下框架：

```
**新闻**：一句话概括

**巴菲特三问**：
- 护城河影响：增强 / 减弱 / 无关
- 管理层信号：正面 / 负面 / 中性
- 盈利质量：改善 / 恶化 / 无变化

**对持仓意义**：具体建议，1-2句
```

**噪音过滤**（直接略过，不分析）：
- "XX只个股突破年线 / 跨越牛熊分界线"
- "XX家公司公布年报"
- "XX行业资金流向日报"
- 超过15天的旧新闻（公告类除外）
