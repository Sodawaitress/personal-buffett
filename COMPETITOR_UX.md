# 竞品 UX 分析报告
> 对标 Yahoo Finance / TradingView / Seeking Alpha / Robinhood / StockCharts
> 生成日期：2026-04-06

---

## 执行摘要

5个平台都有各自的设计优势，但几乎所有平台都有相同的盲区：**缺乏清晰的市场情绪指标**，**多股对比弱**，**批量操作不成熟**。这三个方向是「私人巴菲特」的差异化机会。

---

## 1. Yahoo Finance

**核心理念**：信息民主化 + 个性化并重

**亮点：**
- 可配置仪表板（添加/移除/重排股票卡片）
- 新闻情境化：新闻卡片里直接标注相关股票代码和价格
- 主题聚合：基于阅读历史生成"新闻集群"
- 25+ 图表类型 + 100+ 指标

**不足：**
- 缺乏多股对比（需要开多个标签页）
- 情绪指标不明显
- 信息过载，无限滚动容易分心

**对我们的启发：** 新闻卡片标注相关股票这个设计很好，首页信息流里每条新闻可以标「影响：NVDA↑ 三一重工↓」。

---

## 2. TradingView

**核心理念**：专业交易者工具，深度定制

**亮点：**
- **Sections 分组**：右键创建分组，按市场/策略分类
- **颜色编码**：旗帜标记高优先级股票
- **Advanced View**：自定义列（价格、财务、表现可任意组合）
- **Compare Tool**：2+ 股票价格图表叠加对比
- Quick Actions：不离开列表就能执行操作

**不足：**
- 学习曲线陡，新手容易迷失
- 批量操作设计未找到
- 对基本面分析不够深

**对我们的启发：** 分组 + 颜色标记这两个功能值得加进 `/watchlist` 页。

---

## 3. Seeking Alpha

**核心理念**：研究驱动，定量+定性平衡

**亮点：**
- **Custom View**：100+ 指标，完全自定义显示列，多视图保存切换
- 综合评分系统（估值/增长/盈利/动量/修正 5个维度）
- 投资组合笔记 + 自定义提醒
- 行业分布交互图（直观看资产集中度）

**不足：**
- 没有原生多股并排对比
- 批量编辑不清晰
- 100+ 指标对普通用户压力大

**对我们的启发：** 评分系统就是我们的巴菲特评级（A/B+/B/C/D），但可以在卡片上加"5维雷达图"的迷你版。

---

## 4. Robinhood

**核心理念**：移动优先，极简，情感化设计

**亮点：**
- **卡片扩展动画**：点击卡片从原位置动画展开为全屏
- **背景色传达状态**：市场开盘白色，盘后灰色；涨绿跌红全屏背景
- 新闻卡片左滑消除（像处理邮件一样管理信息）
- 极简下单流程：一键操作

**不足：**
- 无多股对比（依赖第三方工具）
- 分析功能几乎为零
- 基本面数据极少

**对我们的启发：** 情感化设计值得借鉴——巴菲特评级出来的瞬间应该有一个"仪式感"动画。

---

## 5. StockCharts

**核心理念**：图表集合管理，技术分析优先

**亮点：**
- **4 种视图**：全尺寸图表 / 迷你图快速浏览 / 深度单股多时间框架 / 表格对比
- 与扫描工具集成（可对列表跑条件筛选）
- CandleGlance 迷你图一屏看 20 只

**不足：**
- UI 陈旧，移动端体验差
- 无批量操作
- 无社交/社区功能

**对我们的启发：** 「我的选股」页面可以加一个「缩略图模式」，一屏看 20 只股票的评级+迷你走势。

## 横向对比

### Watchlist 功能

| 特性 | Yahoo | TradingView | Seeking Alpha | Robinhood | StockCharts |
|------|:-----:|:-----------:|:-------------:|:---------:|:-----------:|
| 多个列表 | ✓ | ✓ | ✓ | × | ✓ |
| 分组/分段 | 基础 | ★ 高级 | 基础 | × | × |
| 颜色标记 | × | ✓ | × | × | × |
| 自定义列 | ✓ | ★ | ★ | × | ✓ |
| 批量操作 | 弱 | 弱 | 弱 | × | × |
| 多股对比 | × | ★ | × | × | 表格 |

### 单股详情深度

```
深度：Seeking Alpha > Yahoo Finance > TradingView > Robinhood > StockCharts
技术：TradingView > StockCharts > Yahoo > Seeking Alpha > Robinhood
体验：Robinhood > TradingView > Yahoo > Seeking Alpha > StockCharts
```

### 共同盲区（机会）

1. **市场情绪指标** — 所有平台都没有做好看涨/看跌比例的可视化
2. **AI 解读** — 没有平台用 LLM 生成自然语言分析信（这是我们的核心差异化）
3. **批量操作成熟度** — 全行业都很弱

## 对「私人巴菲特」的设计建议

### 建议 1 · 新闻情境化标注（参考 Yahoo Finance）
每条新闻卡片标注「影响股票」：
```
[三一重工] 工程机械出口超预期 → 影响: 三一重工↑ 中联重科↑
```
→ **新 User Story 候选**：新闻卡片标注关联自选股

### 建议 2 · 多股对比视图（参考 TradingView）
`/portfolio` 页面的对比表格，支持自定义列（评级/价格/涨跌/PE/资金流）
→ **已有 US-07**，但对比表格的细节设计可以参考 TradingView Advanced View

### 建议 3 · 迷你图模式（参考 StockCharts CandleGlance）
「我的选股」页面加切换按钮：卡片模式 ↔ 缩略图模式
→ **新 User Story 候选**：Watchlist 缩略图视图

### 建议 4 · 情感化分析结果呈现（参考 Robinhood）
巴菲特信生成完成时，加一个"信封打开"的动画仪式感
→ **小改动**，可以直接做

### 建议 5 · 5 维评分雷达图（参考 Seeking Alpha 评分体系）
单股详情页侧边栏加迷你雷达图：护城河/管理层/估值/资金/宏观
→ **新 User Story 候选**：巴菲特++评分可视化
